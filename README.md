# File Organizer

A simple Python script that automatically organizes files in a selected directory into folders based on their file types.

## Features

- Organizes files by extension
- Creates folders automatically if they don't exist
- Supports:
  - Images
  - Documents
  - Videos
  - Others
- Logs all file movements and errors to `organizer.log`

## Requirements

- Python 3.x
- No external libraries required

## How to Run

1. Open `file_organizer.py`.
2. Update the `DIRECTORY` variable with the folder you want to organize.
3. Run the script:

```bash
python file_organizer.py
```
